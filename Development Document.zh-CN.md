### **2_1. 功能需求**

<center>
表1 功能模块描述

| **功能模块** | **功能** | **描述** | **优先级** |
|:---:|---|---|:---:|
| **用户** | 注册 | 使用邮箱注册 | 高 |
| | 登录 | 使用密码或邮箱验证登录 | 高 |
| | 认证 | 基于JWT的会话管理以保护API | 高 |
| | 个人资料管理 | 更新头像、昵称等 | 中 |
| | 仓库权限 | 设置仓库可见性（公开/私有）并管理协作者 | 高 |
| **仓库** | 创建仓库 | 使用名称、描述、可见性创建仓库 | 高 |
| | 仓库列表 | 列出拥有的、贡献的、标星的仓库，支持分页和搜索 | 高 |
| | 仓库设置 | 修改描述、可见性、删除仓库 | 中 |
| | 分支管理 | 创建/切换/删除分支（Git风格） | 低（可选） |
| | 提交历史 | 显示提交记录，每个提交的更改文件列表 | 中 |
| **文件** | 上传（拖拽&文件夹） | 从本地机器拖拽文件/文件夹上传 | 高 |
| | 上传（文件选择器） | 通过系统文件选择器上传，支持多选 | 高 |
| | 分块上传 | 自动将大文件分割成块，支持断点续传 | 中 |
| | 目录管理 | 在仓库内创建、重命名、删除目录 | 高 |
| | 文件管理 | 重命名、移动、删除文件 | 高 |
| | 文件元数据 | 存储文件大小、MIME类型、上传者、上传时间、最后修改时间 | 高 |
| | 下载 | 下载单个文件；可选下载多个文件或整个目录为ZIP存档 | 高（单个文件）/ 中（批量） |
| **预览** | 文本/代码预览 | 基于扩展名的语法高亮，行号 | 高 |
| | 图像预览 | 内联显示图像，支持缩放 | 高 |
| | PDF预览 | 嵌入PDF.js查看器，支持分页、缩放、下载 | 高 |
| | Office文档预览 | 将Word/Excel/PPT转换为PDF或HTML预览 | 中 |
| | 音频/视频播放 | 内置播放器支持常见格式（MP4, MP3, WebM） | 中 |
| | 目录树 | 在左侧显示仓库文件夹结构，支持折叠/展开 | 高 |
| | 差异比较 | 显示文本文件两个版本之间的差异 | 低（可选） |
</center>
